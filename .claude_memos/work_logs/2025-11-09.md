#### 2. 거리 기반 충돌 감지

**구현:**
```typescript
const distance = Math.sqrt(dx*dx + dy*dy + dz*dz)
if (distance < obstacle.collisionRadius) {
  window.gameFailed = true
}
```

**장점:**
- ✅ 간단한 구현 (10줄)
- ✅ 성능 최적 (150개 장애물도 60 FPS)
- ✅ 조정 가능 (collisionRadius)

**현재 구현:** LocalStorage

**장점:**
- ✅ 즉시 작동 (백엔드 불필요)

- ✅ MVP 출시 가능

**결론:**
- MVP는 LocalStorage로 충분
- 백엔드는 선택적 (Phase 5)
