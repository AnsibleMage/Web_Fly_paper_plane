## 🎓 학습 내용

### 1. formatTime 함수 패턴
**MM:SS.mmm 포맷:**
```typescript
const formatTime = (milliseconds: number): string => {
  const time = Math.max(0, milliseconds);
  const totalSeconds = Math.floor(time / 1000);
  const minutes = Math.floor(totalSeconds / 60);
  const seconds = totalSeconds % 60;
  const ms = time % 1000;

  return `${String(minutes).padStart(2, '0')}:${String(seconds).padStart(2, '0')}.${String(ms).padStart(3, '0')}`;
};
```

**핵심:**
- `Math.max(0, ...)` 음수 방어
- `padStart(N, '0')` 제로 패딩
- 밀리초는 3자리

### 2. formatTimeDifference 패턴
**±X.XXX 포맷:**
```typescript
const formatTimeDifference = (diffMs: number): string => {
  const diffSeconds = diffMs / 1000;
  const absoluteDiff = Math.abs(diffSeconds);
  const formatted = absoluteDiff.toFixed(3);

  if (diffSeconds < 0) return `-${formatted}`;
  else if (diffSeconds > 0) return `+${formatted}`;
  else return `±${formatted}`;
};
```

**핵심:**
- `Math.abs()` 절댓값
- `toFixed(3)` 소수점 3자리
- 부호 처리 (-, +, ±)

### 3. 동적 색상 결정 패턴
```typescript
const getSpeedColor = (speed: number): string => {
  if (speed < 50) return 'text-blue-500';

```

**핵심:**
- 임계값 기반 분기
