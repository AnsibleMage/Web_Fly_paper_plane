### 2. 충돌 감지 방식

**구현:**
```typescript
// 거리 기반 충돌 (AABB 대신)
const dx = plane.x - obstacle.x
const dy = plane.y - obstacle.y
const dz = plane.z - obstacle.z
const distance = Math.sqrt(dx*dx + dy*dy + dz*dz)

if (distance < obstacle.collisionRadius) {
  window.gameFailed = true
}
```

**장점:**
- ✅ 간단한 구현 (10줄)
- ✅ 성능 최적 (150개 장애물도 60 FPS)
- ✅ 조정 가능 (collisionRadius)

**단점:**
- ❌ 정확도 다소 부족 (구형 충돌)
- ✅ 게임 플레이에는 문제 없음
